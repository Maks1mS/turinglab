# turinglab

## Installation

```bat
pip install git+https://github.com/Maks1mS/turinglab.git
```

## Develop

```bat
python setup.py develop
```

## Usage

```bat
turinglab <input_file> <input_string> <output_file>
```

### Example

```bat
turinglab 0to1to0.txt 101 out.docx
```
